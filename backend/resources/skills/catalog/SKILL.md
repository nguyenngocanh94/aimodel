# Catalog Skill

Gọi skill này để nhận bảng kê nhanh tất cả workflow có thể kích hoạt trong hệ thống.
Mỗi entry gồm: slug, name, nl_description.

Dùng để:
- Trả lời câu hỏi "có workflow nào cho X không?"
- So sánh brief với catalog trước khi gọi compose-workflow
- Liệt kê lại danh sách khi người dùng hỏi
